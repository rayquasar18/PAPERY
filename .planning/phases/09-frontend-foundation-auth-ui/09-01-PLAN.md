# Plan 09-01: Next.js 16 Project Scaffold & Core Configuration

## Goal

Create the Next.js 16 + React 19 frontend project with TypeScript strict mode, Tailwind CSS v4, pnpm, and shadcn/ui initialization. This is the foundational scaffold that all other plans depend on.

## Requirement Coverage

| Requirement | Coverage |
|-------------|----------|
| FRONT-01 | Next.js 16 + React 19 App Router setup with TypeScript strict mode |

## Tasks

<task id="09-01-01">
<title>Scaffold Next.js 16 project with pnpm</title>
<read_first>
- .planning/phases/09-frontend-foundation-auth-ui/09-RESEARCH-nextjs-scaffold.md (scaffold steps, folder structure, tsconfig)
- .planning/phases/09-frontend-foundation-auth-ui/09-CONTEXT.md (decisions D-23, D-24)
</read_first>
<action>
1. Create `frontend/.npmrc` with content:
   ```
   shamefully-hoist=true
   strict-peer-dependencies=false
   ```
2. Run `pnpm create next-app@latest frontend --typescript --tailwind --eslint --app --src-dir --import-alias "@/*" --use-pnpm` from project root (or scaffold manually if interactive prompts block).
3. Verify `frontend/package.json` has `next@^16`, `react@^19`, `react-dom@^19`.
4. Update `frontend/tsconfig.json` to ensure:
   - `"strict": true`
   - `"target": "ES2017"`
   - `"moduleResolution": "bundler"`
   - `"paths": { "@/*": ["./src/*"] }`
   - `"plugins": [{ "name": "next" }]`
5. Create `frontend/.env.example` with:
   ```
   NEXT_PUBLIC_BACKEND_API_URL=http://localhost:8000
   NEXT_PUBLIC_API_VERSION=v1
   BACKEND_API_URL=http://localhost:8000
   API_VERSION=v1
   ```
6. Create `frontend/.env.local` copying from `.env.example` (will be gitignored).
</action>
<acceptance_criteria>
- `frontend/package.json` exists and contains `"next"` with version `^16`
- `frontend/tsconfig.json` contains `"strict": true`
- `frontend/tsconfig.json` contains `"@/*": ["./src/*"]`
- `frontend/.npmrc` contains `shamefully-hoist=true`
- `frontend/.env.example` contains `NEXT_PUBLIC_BACKEND_API_URL`
- `pnpm --dir frontend run build` completes without errors
</acceptance_criteria>
</task>

   ```
   ├── i18n/
   └── locale/
   ```
2. Update `frontend/src/app/layout.tsx` as the absolute root layout:
   ```typescript
   import type { Metadata } from 'next';
   import { Inter } from 'next/font/google';
   import './globals.css';

   const inter = Inter({
     subsets: ['latin', 'vietnamese'],
     variable: '--font-sans',
     display: 'swap',
   });

   export const metadata: Metadata = {
     title: 'PAPERY',
     description: 'AI-powered document intelligence platform',
   };

   export default function RootLayout({
     children,
   }: {
     children: React.ReactNode;
   }) {
     return children;
   }
   ```
   NOTE: The `<html>` and `<body>` tags will be in the `[locale]/layout.tsx` (Plan 09-02) since they need the `lang` attribute from locale.
