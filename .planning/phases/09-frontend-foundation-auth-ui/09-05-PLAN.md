## Threat Model

<threat_model>
| Threat | Severity | Likelihood | Mitigation |
|--------|----------|------------|------------|
| XSS via form inputs | HIGH | LOW | React auto-escapes JSX output; Zod validates all inputs before submission |
| CSRF on login/register forms | MEDIUM | LOW | HttpOnly cookies + SameSite=Lax; backend validates CORS origin |
| Password in URL params | HIGH | NONE | Passwords only in POST body; tokens in URL are one-time-use backend tokens |
| Brute force login attempts | MEDIUM | MEDIUM | Backend rate limiting (Phase 6); frontend shows loading state to slow UX |
| Token in URL for verify/reset | MEDIUM | LOW | Tokens are time-limited and single-use on backend side |
</threat_model>
