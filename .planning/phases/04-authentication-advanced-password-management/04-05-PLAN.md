        provider = _get_github_provider()

        user_info = await provider.get_user_info(access_token)
        user = await auth_service.oauth_login_or_register(db, user_info)

    # Issue JWT token pair

    # Set cookies on redirect response

This exactly mirrors the working `google_callback` pattern (lines 455-472), with these provider-specific substitutions:
- `_get_google_provider()` → `_get_github_provider()`
- `"Google OAuth callback failed"` → `"GitHub OAuth callback failed"`

The six operations in the try block are:
1. `_get_github_provider()` — create provider instance (may raise NotFoundError)
2. `provider.get_access_token(code)` — exchange authorization code for provider access token
3. `provider.get_user_info(access_token)` — fetch user info from GitHub API
4. `auth_service.oauth_login_or_register(db, user_info)` — create or link user account

The four operations after the try block are:
5. `create_token_pair(user.uuid)` — issue JWT access + refresh tokens, defining `access_jwt` and `refresh_jwt`
6. `decode_token(refresh_jwt)` + `register_token_in_family(...)` — register refresh token in family for replay detection
7. `track_user_family(user.uuid, refresh_payload.family)` — track user→family mapping
8. `_set_auth_cookies(redirect, access_jwt, refresh_jwt)` — set HttpOnly cookies on redirect response

Add this test method to `TestOAuthCallbackSuccess`:

```python
    async def test_github_callback_provider_exception_redirects_with_failed_error(
        self, async_client: AsyncClient
    ):
        """GitHub provider HTTP failure redirects to login with oauth_failed."""
        with (
            patch(
                "app.api.v1.auth.validate_oauth_state",
                new_callable=AsyncMock,
                return_value=True,
            ),
            patch("app.api.v1.auth._get_github_provider") as mock_provider_factory,
            patch("app.api.v1.auth.settings") as mock_settings,
        ):
            mock_settings.FRONTEND_URL = "http://localhost:3000"
            mock_settings.GITHUB_CLIENT_ID = "configured"
            mock_settings.GITHUB_CLIENT_SECRET = "configured"
            mock_settings.OAUTH_REDIRECT_BASE_URL = "http://localhost:8000"

            mock_provider = MagicMock()
            mock_provider.get_access_token = AsyncMock(
                side_effect=ValueError("Token exchange failed")
            )
            mock_provider_factory.return_value = mock_provider

            response = await async_client.get(
                "/api/v1/auth/github/callback",
                params={"code": "bad-code", "state": "valid-state"},
                follow_redirects=False,
            )

        assert response.status_code == 302
        assert "error=oauth_failed" in response.headers.get("location", "")
```
