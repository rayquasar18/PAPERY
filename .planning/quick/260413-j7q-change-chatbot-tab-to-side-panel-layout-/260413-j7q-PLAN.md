# Quick Task 260413-j7q: Change chatbot tab to inline side panel

## Objective

Replace the Sheet (dialog overlay) chat panel with an inline side panel that lives inside the
layout. When toggled open, the chat panel appears on the right and the main content area shrinks
to accommodate it — so the user can interact with both the page content and the chatbot
simultaneously without any overlay or backdrop.

**Files:**

**Action:**

1. Remove all Sheet-related imports (`Sheet`, `SheetContent`, `SheetTitle`, `SheetTrigger`) and
   the `ChatPanel` import — ChatPanel will now be rendered in the layout, not in TopBar.
2. Remove the `useTranslations('Chat')` call (no longer needed here since Sheet title is gone).
3. Import `useSidebarStore` from `@/stores/sidebar-store`.
4. Replace the entire `<Sheet>...</Sheet>` block with a simple toggle button:

```tsx
import { useSidebarStore } from '@/stores/sidebar-store';

// Inside the component:

// In the JSX — replace the Sheet block with:
```

## Task 2: Render inline ChatPanel in layout-client with push/resize behavior

**Files:**
- `frontend/src/app/[locale]/(dashboard)/layout-client.tsx`
- `frontend/src/components/layout/chat-panel.tsx` (minor: add close button)

**Action:**

### layout-client.tsx

1. Import `useSidebarStore` from `@/stores/sidebar-store` and `ChatPanel` from
   `@/components/layout/chat-panel`.
2. Import `X` icon from `lucide-react` and `Button` from `@/components/ui/button`.
3. Read `isChatPanelOpen` and `toggleChatPanel` from the store.
4. Restructure the content area inside `<SidebarInset>` so that below the `<TopBar />`,
   there is a flex row container holding the main content and the chat panel side-by-side.
   The chat panel slides in from the right and the main content shrinks via CSS transition.

Replace the current content structure:

```tsx
<SidebarInset>
  <TopBar />
  <div className="flex flex-1 flex-col">
    <div className="@container/main flex flex-1 flex-col gap-2">
      {children}
    </div>
  </div>
</SidebarInset>
```

With:

```tsx
<SidebarInset>
  <TopBar />
  <div className="flex flex-1 overflow-hidden">
    {/* Main content area — shrinks when chat panel is open */}
    <div
      className="flex flex-1 flex-col min-w-0 transition-all duration-300 ease-in-out"
    >
      <div className="@container/main flex flex-1 flex-col gap-2">
        {children}
      </div>
    </div>

    {/* Inline chat panel — pushes content when open */}
    <div
      className={cn(
        'shrink-0 border-l bg-background transition-all duration-300 ease-in-out overflow-hidden',
        isChatPanelOpen ? 'w-[400px] lg:w-[480px]' : 'w-0 border-l-0'
      )}
    >
      {isChatPanelOpen && (
        <div className="h-full w-[400px] lg:w-[480px]">
          <ChatPanel onClose={toggleChatPanel} />
        </div>
      )}
    </div>
  </div>
</SidebarInset>
```

5. Import `cn` from `@/lib/utils`.

**Key layout decisions:**
- `overflow-hidden` on the flex row prevents the chat panel from overflowing during transition.
- The chat panel wrapper uses fixed width classes (`w-[400px] lg:w-[480px]`) that transition
  to `w-0` when closed, creating the push/resize effect.
- `min-w-0` on the main content area ensures it can shrink below its intrinsic minimum width.
- `border-l` provides a visual separator between main content and chat panel.
- The inner `<div>` with fixed width prevents content from reflowing during the width transition.
- `shrink-0` on the chat panel wrapper ensures it maintains its width and the main content
  is what shrinks.
