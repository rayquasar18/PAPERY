# Quick Task: Separate Chatbot Panel as Independent Floating Overlay

## Objective

Convert the ChatPanel from an inline side panel (that pushes main content) to an independent floating overlay with rounded corners. The panel should float on top of the content area, be completely decoupled from the main content layout, and be openable at any time regardless of the current page.

## Context

The previous quick task (260413-j7q) changed the ChatPanel from a Sheet overlay to an inline side panel that pushes main content via CSS flex layout. The user now wants to revert to a floating overlay approach, but NOT using shadcn Sheet -- instead a custom floating panel with rounded corners that overlays on top of content without pushing it.

### Current State
- `layout-client.tsx`: ChatPanel rendered inline in a flex row, pushes main content when open
- `chat-panel.tsx`: Renders panel content with header + scrollable body
- `sidebar-store.ts`: Zustand store with `isChatPanelOpen`, `toggleChatPanel`, `chatPanelWidth`
- `top-bar.tsx`: Toggle button for chat panel using sidebar store

## Tasks

### Task 1: Convert ChatPanel to floating overlay in layout-client.tsx and add rounded corners to chat-panel.tsx

**Type:** auto

**Files:**
- `frontend/src/app/[locale]/(dashboard)/layout-client.tsx`
- `frontend/src/components/layout/chat-panel.tsx`

**Action:**

1. In `layout-client.tsx`, remove the inline flex row layout for the chat panel. Instead of rendering ChatPanel inside a flex sibling div that pushes content, render it as a `fixed` or `absolute` positioned overlay that floats on top of the content area. The main content should always take full width regardless of chat panel state.

   Specific changes to `layout-client.tsx`:
   - Remove the outer `<div className="flex flex-1 overflow-hidden">` wrapper that creates the flex row
   - Restore the main content to be a direct child of `SidebarInset` (after TopBar) without flex row constraints
   - Add the ChatPanel as a positioned overlay element. Use `fixed` positioning with `bottom-4 right-4` (or similar) so it floats in the bottom-right corner of the viewport. Give it dimensions like `w-[400px] lg:w-[480px] h-[600px] max-h-[80vh]`.
   - Wrap the overlay in a conditional render based on `isChatPanelOpen`
   - Add `z-50` to ensure it floats above content
   - Add open/close animation: use `transition-all duration-300` with scale/opacity transform for smooth appear/disappear

2. In `chat-panel.tsx`, add rounded corners to the outer container:
   - Change the outer div className from `"flex h-full flex-col bg-background"` to include `rounded-xl border shadow-xl` for the floating appearance
   - Add `overflow-hidden` to ensure content respects the rounded corners
   - Ensure the `onClose` X button is always visible (remove the conditional `{onClose &&` -- always show the close button since the floating panel always needs a way to close)
